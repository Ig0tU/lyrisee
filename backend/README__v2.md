
# The Lyric Weaver

This project is an AI-powered kinetic typography generator that brings lyrics to life. It uses a Python backend to process an audio file and a Three.js frontend to create a dynamic, 3D visual representation of the lyrics.

This application was co-created with the Gemini AI assistant.

## Project Structure

- `audio_processor.py`: A Python script that uses `demucs` for source separation, `stable-ts` for transcription and word-level timestamping, and `librosa` for beat detection. It takes an audio file as input and generates `lyric_data.json`.
- `requirements.txt`: A list of Python dependencies required for the backend.
- `public/`: A directory containing the frontend web application.
  - `index.html`: The main Three.js application that visualizes the data from `lyric_data.json`.
  - `lyric_data.json`: The data file generated by the Python script (will be moved here by the run script).
  - `DarkNights-80K.mp3`: The audio file (will be copied here by the run script).
- `run.sh`: A shell script to automate the setup and execution process.

## How to Run

### Prerequisites

- Python 3.x
- `pip` (Python package installer)
- `ffmpeg`: A command-line tool for audio/video processing. You can install it via Homebrew (`brew install ffmpeg`) on macOS or your system's package manager on Linux.

### Steps

1.  **Make the run script executable:**
    Open your terminal in the project directory and run:
    ```bash
    chmod +x run.sh
    ```

2.  **Execute the run script:**
    This script will handle everything from installing dependencies to providing the final command to view the result.
    ```bash
    ./run.sh
    ```
    The script will:
    a. Install the required Python packages.
    b. Run the audio processor on `DarkNights-80K.mp3`. This step is computationally intensive and may take several minutes.
    c. Move the generated `lyric_data.json` and the audio file into the `public/` directory.
    d. Print the final instructions to view the application.

3.  **Start the local web server:**
    As instructed by the `run.sh` script, open a **new terminal window**, navigate to the project directory, and run the following command to serve the `public` directory:
    ```bash
    python3 -m http.server -d public 8000
    ```

4.  **View the result:**
    Open your web browser and navigate to:
    [http://localhost:8000](http://localhost:8000)

    Click "Play" to experience the lyric visualization.
